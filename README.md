# Landingpage

Finales Projekt aus dem Fundamentals-Kurs von The Odin Project.
Ziel war es, eine Landingpage auf Basis einer Vorlage mit HTML und CSS nachzubauen.

## Features

- Header mit Logo und Navigationslinks
- Hero-Bereich mit Titel, Subtext und Call-to-Action-Button
- Layout mit Flexbox
- Grundlegendes Styling mit Farben, Abstaenden und Typografie

## Tech Stack

- HTML5
- CSS3 (Flexbox)
- Google Fonts (Roboto)

## Projekt lokal starten

1. Repository klonen
2. `index.html` im Browser oeffnen

## Lernziele

- Strukturierte HTML-Semantik ueben
- Flexbox fuer Layouts einsetzen
- Typische Landingpage-Bausteine nachbauen
- Designvorgaben moeglichst nah umsetzen
